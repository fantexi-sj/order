## 七、打印机功能详解（重点）

### 7.1 为什么这个功能重要？
- 顾客下单后，厨房需要知道做什么菜
- 传统方式是服务员手写或口传，容易出错
- 打印机自动打印小票，厨房直接按单制作，效率高、错误率低

### 7.2 实现方案

#### 方案一：USB/蓝牙小票打印机（适合单店）
- 使用 ` usb` 或 `bluetooth` API 连接打印机
- 调用打印机厂商提供的 SDK
- 常见品牌：芯烨、佳博、商祺

#### 方案二：网络打印机（适合连锁店）
- 打印机连接局域网/互联网
- 调用打印机的 HTTP API
- 支持远程打印

#### 方案三：第三方打印服务（推荐）
- 使用 易联云、飞鹅 等第三方打印服务
- 提供现成的 API 和 SDK
- 支持多平台、多打印机

### 7.3 打印内容格式
```
========== EasyOrder ==========
订单号: 2026041912345
时间: 2026-04-19 12:00
桌号: 03号桌
--------------------------------
🍔 汉堡套餐 x1    ¥28.00
🍟 薯条(大) x1    ¥12.00
🥤 可乐 x2       ¥10.00
--------------------------------
合计: ¥50.00
备注: 汉堡少酱
================================
      请稍候，您的餐正在准备中
```
